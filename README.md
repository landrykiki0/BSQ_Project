# **Bsq**
Ce projet consiste à trouver le plus grand carré dans une map composée de '.' et 'o' où '.' représente un champ libre et 'o' une barrière

# Installation

Voici comment lancer le projet sur votre ordinateur:

Cloner le repository

## Compilation du projet

<pre>
  make
</pre>

## Lancement du projet
<pre>
./bsq map1
</pre>

<pre>
./bsq map2
</pre>
